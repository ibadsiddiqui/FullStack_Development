By using the span element, you can put several elements together, 
and even style different parts of the same element differently.

Nest the word "love" in your "Things cats love" element below within a span element. 
Then give that span the class text-danger to make the text red.

Here's how you would do this with the "Top 3 things cats hate" element:

<p>Top 3 things cats <span class = "text-danger">hate:</span></p>
